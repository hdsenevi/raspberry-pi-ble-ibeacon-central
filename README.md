# Raspberry PI BLE iBeacon central test

## Prerequisites

node

``` 
v10.23.0
```

on raspberry pi

```
sudo apt-get install bluetooth bluez libbluetooth-dev libudev-dev
```

## Instrunctions

install dependancies
```
npm install
```

run
```
sudo node app.js
```
